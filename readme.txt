== Description ==

"Mosque Prayer Timings" plugin is for Mosques' websites. It helps publish Adhan+Iqamah timings of daily prayers, and Khutba+Iqamah timings of Friday prayers at the Mosque.

If the website theme is mobile-enabled, then the prayer timings can be viewed on mobile devices at:
www.yourwebsite.org/#masjid

== Installation ==

1. Upload mosque-prayer-timings plugin folder to /wp-content/plugins/ directory.
1. Activate the plugin through "Plugins" section in WordPress dashboard.
1. Add "Mosque Prayer Timings" widget through "Appearance > Widget" to your wordpress theme's sidebar.
1. Populate widget form with daily and/or friday prayer timings.

== Screenshots ==

1. Populate widget form with prayer timings and click "Save".

== Frequently Asked Questions ==

= Does this plugin make connections to other servers? =


This is a stand-alone plugin and does NOT make connections to other servers.
